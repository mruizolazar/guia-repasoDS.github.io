Introducción al Análisis Exploratorio de datos

Introducción

En 1962, John W. Tukey (Figura) pidió una reforma de las estadísticas en su documento seminal “El futuro del análisis de datos” [Tukey-1962]. Propuso una nueva disciplina científica llamada análisis de datos que incluía la inferencia estadística como un solo componente.

En 1962, John W. Tukey (Figura) pidió una reforma de las estadísticas en su documento seminal “El futuro del análisis de datos” [Tukey-1962]. Propuso una nueva disciplina científica llamada Análisis de datos que incluía la inferencia estadística como un solo componente.

El campo del análisis de datos exploratorios se estableció con el libro de Tukey de 1977, el clásico Análisis de datos exploratorios [Tukey-1977].

Definición

  • El análisis exploratorio de datos (EDA) es el proceso de explorar sus datos, y generalmente incluye examinar la estructura y los componentes de su conjunto de datos, las distribuciones de variables individuales y las relaciones entre dos o más variables.
  • La herramienta más utilizada para el análisis exploratorio de datos es la visualización datos utilizando una representación gráfica de los datos.
  • La visualización de datos es posiblemente la herramienta más importante para el análisis exploratorio de datos porque la información transmitida por la visualización gráfica se puede absorber muy rápidamente y porque generalmente es fácil reconocer patrones en una visualización gráfica.
  • La visualización es siempre apoyada por las medidas de resumen. Medidas que pretenden caracterizar el conjunto con la menor distorsión o perdida de información posible. Se realiza el resumen de la característica en uno o dos números.
    • Ejemplos: media, mediana, moda, desviación estándar, mínimo, máximo etc.

Objetivos

  • Para determinar si hay algún problema con su conjunto de datos.
  • Para determinar si la pregunta que está haciendo puede ser respondida por los datos que tiene.
  • Desarrollar un bosquejo de la respuesta a su pregunta.
    • Ejemplo: ¿Las ciudades del este del país tienen niveles de ozono más altos que las ciudades del oeste?
      • Para responder esta pregunta necesito datos de ozono, ciudades y de las regiones

Tipos de Análisis

  • Análisis univariado: Es la forma más simple de analizar datos donde examinamos cada variable individualmente.
    • Para las características categóricas podemos usar tablas de frecuencia o gráficos de barras que calcularán el número de cada categoría en una variable particular.
    • Para las características numéricas, las gráficas de densidad de probabilidad se pueden usar para observar la distribución de la variable.
  • Análisis multivariado: es un método estadístico utilizado para determinar la contribución de varios factores en un simple evento o resultado.
    • Ejemplos: análisis de componentes principales, análisis factorial, análisis discriminante, análisis de correlación etc.
