Programación con Python para Ciencia de datos

Uso del lenguaje de Programación Python

Python es un lenguaje de programación interpretado cuya filosofía hace hincapié en una sintaxis que favorezca un código legible.  Se trata de un lenguaje de programación multi-paradigma, ya que soporta orientación a objetos, programación imperativa y, en menor medida, programación funcional.

Python se ha convertido en las últimas dos décadas como una herramienta de primera clase para tareas de computación científica, incluido el análisis y la visualización de grandes conjuntos de datos. Principalmente por sus librerías para almacenar, manipular, y obtener información desde los datos. Varios recursos existen para dar soporte al Científico de datos en su tarea de descubrimiento de información relevante. Algunos de estos son:

  • iPython y Jupyter: proporciona un ambiente computacional para científicos de datos usando Python,
  • Numpy: incluye los ndarray para almacenamiento eficiente y manipulación de arrays de datos densos en Python,
  • Pandas: presenta el DataFrame para el almacenamiento eficiente y la manipulación de datos etiquetados/en columnas en Python,
  • Matplotlib: incluye capacidades para una gama flexible de visualizaciones de datos en Python,
  • Scikit-Learn: para implementaciones eficientes y limpias de código en Python de los más importantes y reconocidos algoritmos de machine learning,
  • y otras herramientas relacionadas.

En esta sección conoceremos y practicaremos con estas herramientas. Pero antes vamos a necesitar conocer la plataforma computacional Jupyter notebook que usaremos para desarrollar nuestros proyectos de Ciencia de datos.
